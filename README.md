## Ecoando Para Quem Ama
Projeto desenvolvido para a disciplina de Inovações para Engenharia da UFABC

Este projeto teve como objetivo o desenvolvimento de uma startup de base tecnológica voltada para a criação de soluções inovadoras na área da saúde e assistência ao idoso. A proposta consistiu no desenvolvimento de um dispositivo inteligente para detecção de quedas, projetado para aumentar a segurança e a autonomia de pessoas idosas, contribuindo para uma resposta mais rápida em situações de emergência.

O sistema foi desenvolvido utilizando sensores de aceleração e giroscópio, capazes de monitorar continuamente os movimentos do usuário e identificar padrões associados a quedas ou situações de risco. A integração dos sensores com uma plataforma baseada em Arduino permitiu a coleta e o processamento dos dados em tempo real, possibilitando a implementação de algoritmos para análise e classificação dos eventos detectados.

Como diferencial, o dispositivo conta com um sistema automatizado de comunicação que, ao identificar uma possível queda, envia notificações instantâneas via WhatsApp para familiares, cuidadores ou responsáveis previamente cadastrados. Dessa forma, o projeto busca reduzir o tempo de resposta em emergências e aumentar a segurança dos usuários, especialmente daqueles que vivem sozinhos ou possuem mobilidade reduzida.

Além do desenvolvimento técnico do protótipo, o projeto envolveu etapas de modelagem de negócios, validação da proposta de valor, análise de mercado e definição de estratégias para inserção do produto no setor de tecnologias assistivas. A iniciativa permitiu a aplicação prática de conceitos de eletrônica embarcada, programação, Internet das Coisas (IoT), empreendedorismo e inovação tecnológica voltada para a saúde.
